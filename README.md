# DeluxMed Health Group Experience

Professional healthcare website for DeluxMed Health Group, LLC. This project provides an integrated platform for primary care and behavioral health services.

## Core Features

- **Integrated Care**: Unified approach to primary and behavioral health.
- **HIPAA-Compliant Telehealth**: Secure virtual consultations.
- **Patient Portal**: Access to secure medical records and booking.
- **Responsive Design**: Optimized for all devices.

## Tech Stack

- **Frontend**: React 18 with Vite
- **Styling**: Tailwind CSS & Shadcn/UI
- **Animations**: Framer Motion
- **State Management**: TanStack Query (React Query)
- **Forms & Validation**: React Hook Form & Zod
- **Routing**: React Router DOM v6
